# Security Policy

## Supported versions

`confluence-to-notion` is pre-1.0 and under active development. Only the
latest release on `main` is supported.

| Version | Supported |
| --- | --- |
| latest | ✓ |
| older | ✗ |

## Reporting a vulnerability

Please report security issues **privately** — do not open a public GitHub
issue.

Email: **sunny.dev.js@gmail.com**

Include:

- A description of the vulnerability and its impact.
- Steps to reproduce or a proof-of-concept.
- Your preferred disclosure timeline.

You can expect an acknowledgement within 7 days. Fixes are published as
regular releases once the root cause is understood; the changelog will note
the fix without naming the reporter unless asked to.
